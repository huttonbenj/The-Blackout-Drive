## Data Privacy

- Blackout Protocol (default ON) enforces Network Lock + Encrypt Chat History simultaneously
- The Blackout Drive makes **zero network connections** when Network Lock is on (default)
- If Network Lock is disabled, the only request is a GET to the content catalog (a JSON file list)
- No telemetry, no analytics, no cookies, no user tracking
- All chat data is stored locally on the USB drive — never uploaded
- Chat history is AES-256-GCM encrypted when Encrypt Chat History is enabled
- User files in Encrypted Storage can be stored in Unlocked (standard) or Locked (AES-256-GCM encrypted) directories
- A Single Ecosystem Key (master password) governs all cryptographic operations
- No personal information is collected or transmitted under any circumstance
