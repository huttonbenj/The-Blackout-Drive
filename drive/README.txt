  HOW TO START:

  WINDOWS USERS:
    Double-click "Start (Windows).bat"

  MAC USERS:
    Double-click "The Blackout Drive" app.

  LINUX USERS (Experimental — not yet fully tested):
    Right-click "Start (Linux).sh" and select
    "Run as a program" (or run it from terminal).

  That's it. Your browser will open with the AI
  chat interface. Keep the terminal window open
  while using it.

═══════════════════════════════════════════════════

  FIRST TIME USERS:

  Windows: If SmartScreen says "Windows protected
  your PC," click "More info" then "Run anyway."
  This only happens once.

  Mac: If macOS says "can't be opened because it's
  from an unidentified developer," go to:
    System Settings → Privacy & Security → "Allow Anyway"
  Then try again. This only happens once.

═══════════════════════════════════════════════════

  HOW TO STOP:

  Just close the browser tab. The system will
  shut everything down automatically.

  You can also close the terminal/command window
  directly if you prefer.

  Wait for the terminal window to close, then
  safely eject the drive.

  WHAT'S INCLUDED:

  • BEACON — Offline AI, powered by Qwen3 (Alibaba Cloud) running via Ollama. Ask anything: medicine, engineering, law, survival, cybersecurity, and more.
  • LIBRARY — Offline reference books, manuals, and guides with a built-in reader.
  • WORKSPACE — Personal file storage. Unlocked (open access) or Locked (password-protected, AES-256-GCM encrypted).
  • TOOLS — 6 interactive offline utilities: Ham Radio, Tactical Navigator, Cipher Studio, Survival Calculators, Medical Timers, and Prep Checklists.
  • COMMS — Mesh communications via LoRa radio, built on the open-source Meshtastic protocol (meshtastic.org). Off-grid messaging, live node tracking, and AI dispatch. (Requires Mesh Bundle or a Heltec V3 LoRa radio.)
  • PROMPTS — 100+ curated ready-made queries across 11 knowledge domains.
  • CHATS — Save, encrypt, export, and revisit past conversations.
  • BLACKOUT PROTOCOL — One-click master switch that enforces Network Lock and encrypts all chat history.

═══════════════════════════════════════════════════

  MASTER PASSWORD:

  The master password is optional. You'll be prompted
  to create one the first time you enable encryption —
  either by turning on "Encrypt Chat History" in
  Settings, or by uploading a file to the Locked tab
  in Workspace. This one password protects:

  • Your encrypted conversations
  • Your locked files (in Workspace → Locked)
  • Exported conversation backups

  ⚠ There is no password recovery. If you forget
  your password, you can reset it — but all
  encrypted data will be permanently deleted.
  BEACON, the Library, Prompts, and unlocked files
  are NOT affected by a reset.

  You can change your password anytime:
  Settings → Data → Change Master Password
  (this re-encrypts everything — nothing is lost)

  REQUIREMENTS:
    • macOS 11+ (Apple Silicon or Intel)
    • Windows 10/11 with a dedicated GPU (NVIDIA or AMD)
    • Linux (64-bit, experimental — not yet fully tested) with a dedicated GPU (NVIDIA or AMD)
    • 8GB RAM minimum (16GB recommended)
    • USB 3.0+ port (recommended for best performance)

  No internet connection required. Ever.

  NOTE: On Windows computers without a dedicated GPU
  or with less than 8GB RAM, BEACON AI will be disabled but
  all other features (Library, COMMS, Workspace, Ham Radio)
  work normally.

  PERFORMANCE DISCLAIMER:
  BEACON is a fully offline AI — it runs entirely on
  your computer, not in the cloud. Because of this,
  speed depends on your computer's hardware (processor,
  RAM, and GPU). The first response after each launch
  may take longer while the AI loads into memory.
  Minimum required RAM: 8GB.
  Recommended for faster performance:
  16GB RAM or higher with a dedicated GPU.

  AI MODEL: BEACON is powered by Qwen3, an open-source
  language model developed by Alibaba Cloud (Apache 2.0
  License), running via Ollama (MIT License). We did not
  create the AI model — we packaged and configured it
  to run offline from this drive.
