═══════════════════════════════════════════════════

  HOW TO START:

  WINDOWS USERS:
    Double-click "Start (Windows).bat"

  MAC USERS:
    Double-click "The Blackout Drive" app.

  LINUX USERS:
    Right-click "Start (Linux).sh" and select
    "Run as a program" (or run it from terminal).

  That's it. Your browser will open with the AI
  chat interface. Keep the terminal window open
  while using it.

═══════════════════════════════════════════════════

  FIRST TIME USERS:

  Windows: If SmartScreen says "Windows protected
  your PC," click "More info" then "Run anyway."
  This only happens once.

  Mac: If macOS says "can't be opened because it's
  from an unidentified developer," go to:
    System Settings → Privacy & Security → "Allow Anyway"
  Then try again. This only happens once.

═══════════════════════════════════════════════════

  HOW TO STOP:

  Just close the browser tab. The system will
  shut everything down automatically within
  about a minute.

  You can also close the terminal/command window
  directly if you prefer.

  Wait for the terminal window to close, then
  safely eject the drive.

═══════════════════════════════════════════════════

  MASTER PASSWORD:

  The first time you use the drive, you'll create
  a master password. This one password protects:

  • Your encrypted conversations
  • Your locked files (in Library → My Uploads)
  • Exported conversation backups

  ⚠ There is no password recovery. If you forget
  your password, you can reset it — but all
  encrypted data will be permanently deleted.
  BEACON, the Library, Prompts, and unlocked files
  are NOT affected by a reset.

  You can change your password anytime:
  Settings → Data → Change Master Password
  (this re-encrypts everything — nothing is lost)
