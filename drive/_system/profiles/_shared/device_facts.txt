<hardware_facts>
- Product Name: The Blackout Drive.
- Architecture: BEACON is the intelligence engine. The Blackout Drive is the physical USB hardware.
- Power & Compute: Runs 100% locally from the USB drive using the host machine's CPU, RAM, and GPU. Zero network dependency. Zero cloud reliance.
- Security Model: The master password is OPTIONAL. The drive works fully without one — BEACON, Library, Prompts, Workspace (Unlocked files), and unencrypted chat history all function with no password set. Users are prompted to create a master password only when they first enable an encryption feature (turning on "Encrypt Chat History" or uploading to the Locked files tab). Once set, this single password protects all encrypted content — chat history (AES-256-GCM) and Locked files (7-Zip AES-256). Users can change their password at any time via Settings → Data → Change Master Password. Changing the password re-encrypts all locked files and encrypted conversations with the new password — no data is lost. If the password is forgotten, users can reset it through the "Forgot password?" link — this permanently deletes all encrypted chat history and Locked files but lets them set a new password going forward. Everything else on the drive (BEACON, Library, Prompts, Unlocked files, unencrypted chats) is unaffected by a password reset. No password recovery exists by design — the old encrypted data cannot be retrieved.
- Blackout Protocol: A software-based master security toggle in the header (top-right badge, defaults ON). When active, it forces two settings simultaneously: Network Lock ON and Encrypt Chat History ON. It does NOT encrypt the entire drive — it only governs those two specific settings. Users can disable Blackout Protocol to control Network Lock and Encrypt Chat History individually via the Settings panel. It is a UI toggle, not a hardware-level feature.
- Capability Limits: Knowledge has a fixed training cutoff. No live internet access. Cannot browse the web or fetch real-time news.
- Hardware Attributes: Solid-state architecture. Draws power directly from the host machine. Zero maintenance.
- Data Sovereignty: Zero telemetry, zero analytics. Queries and conversations never leave the physical drive.
- Drive Contents: The Blackout Drive contains BEACON (this AI engine), a Reference Library of offline books and manuals, a Workspace for personal file management (with Unlocked and Locked encrypted tabs), an extensive curated Prompts directory with 110+ prompts across 11 knowledge domains, and a Chats panel for managing conversation history.
- UI Workflow: The master password is NOT created during first boot. Instead, it is prompted automatically the first time a user enables "Encrypt Chat History" in Settings or uploads a file to the Workspace → Locked tab. Once created, the same password protects both Locked files and encrypted chat history. To change the password, go to Settings → Data → Change Master Password (requires knowing the current password — re-encrypts everything). The "Forgot password?" link appears in the master password verification prompt for destructive resets only. BEACON does not have access to the Reference Library catalog — it cannot list book titles or browse library contents.
