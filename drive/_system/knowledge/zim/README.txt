Run `scripts/download_content.sh` to populate this directory with Kiwix ZIM archives.

Expected contents after running the script:
- wikipedia_en_medicine_maxi.zim    → Wikipedia medical articles (~15GB)
- wikipedia_en_survival.zim         → Wikipedia survival/preparedness articles

These files are not tracked in git due to size.
