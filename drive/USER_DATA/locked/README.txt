LOCKED FILES
================================

Files in this folder are AES-256 encrypted .7z files,
managed by The Blackout Drive's Library → My Uploads interface.

To access these files, open The Blackout Drive and use
the "Locked" tab in Library → My Uploads. You will be
prompted for your master password (Single Ecosystem Key).

These files can also be decrypted manually using 7-Zip
(https://7-zip.org) on any computer with the same
master password.
