YOUR FILES — UPLOADED CONTENT
==============================

Place any personal documents here (EPUBs, PDFs, images, etc.) to
access them from The Blackout Drive's Library → My Uploads interface.

Files in this folder are stored unencrypted on the USB drive.
For sensitive documents, use the "Locked" tab instead.

All files in this directory are protected during drive updates —
they will never be overwritten or deleted by the flash process.
